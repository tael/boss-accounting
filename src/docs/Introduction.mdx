# 사장님 회계 도우미 🧾

> "사장님이 알면 돈버는 회계" 책 기반 소규모 사업자용 회계 관리 웹 서비스

## 컴포넌트 라이브러리

이 Storybook은 **사장님 회계 도우미**의 모든 UI 컴포넌트를 문서화합니다.

| 카테고리 | 컴포넌트 | 설명 |
|---------|---------|------|
| 공통 컴포넌트 | Button, Input, Modal | 재사용 가능한 기본 UI |
| 대시보드 | MonthSummary, StatCard, InsightCard, GoalWidget, StreakBadge, HealthScoreWidget | 경영 현황 요약 |
| 거래 관리 | TransactionForm | 매출/비용 입력 폼 |
| 세금 계산기 | VatCalculator, LaborCalc | 부가세·인건비 계산 |

## 기술 스택

- **React 19** + **TypeScript**
- **Tailwind CSS 4** (CSS-first 설정)
- **Zustand 5** (상태 관리)
- **Recharts** (차트)

## Storybook AI (MCP)

Storybook이 실행 중일 때 Claude Code는 MCP를 통해 컴포넌트 정보를 자동 참조합니다.

